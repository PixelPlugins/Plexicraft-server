What is Plexicraft?

Plexicraft is a collection of everything! It has: Plexicraft-core, Plexichat, Flexums, Plexigames, Pleximedia

How to start a custom server

Plexicraft allows you to create a custom server, or alternate universe as they call it. Just clone or download the project, then run npm start.

How to connect to your custom server

To connect to your custom server, go to the plexicraft site and enter the following commands into the f12 developer console:

'ws = new WebSocket("ws://localhost:5000");'
